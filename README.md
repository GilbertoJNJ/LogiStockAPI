## Install 
1. In the terminal, clone the project:
```shell script
git clone https://github.com/GilbertoJNJ/Store-API.git
```

2. Enter in the projet diretory:
```shell script
cd ~\storeapi
```

3. Execute the command:
```shell script
mvn spring-boot:run
```
